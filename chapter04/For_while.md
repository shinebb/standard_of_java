조건문과 반복문
=========================

"제어문(flow control statement)"  
조건문 : 조건을 만족할때만 {}를 수행(0~1번)(if, switch)  
반목문 : 조건을 만족하는 동안 {}를 수행(0~n번)(for, while)


반복문 - for문
---------
*****
"조건을 만족하는 동안 블럭{}을 반복 - 반복횟수를 알 때 적합"  

for(int i=1; i<=5; i++){  
* //1부터 5까지 1씩 증가
* //반복될 코드들  

}  
=>  
for((1)초기화; (2)조건식; (4)증감식){
* (3)수행될 문장
* //2-3-4를 계속 반복하다가 2가 false일 때 끝남
  
}
