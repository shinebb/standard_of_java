임의의 정수(or 실수) 만들기
---------
*****
"Math.random() - 0.0과 1.0사이의 임의의 double값을 반환"

- 0.0 <= Math.random() < 1.0
- 0.0 ~ 0.9999..

1~3의 정수를 만들고 싶다면?
: 

1. 각변에 3을 곱한다.  
    0.0 * 3 <= Math.random() * 3 < 1.0 * 3  
    0.0 <= Math.random() *3 < 3.0  
    => (0.0~2.9999...)  
  
2. 각 변을 int형으로 변환한다.  
    (int)0.0 <= (int)(Math.random() * 3) < (int) 3.0  
    0 <= (int)(Math.random() *3) < 3 (0~2)  

3. 각 변에 1을 더한다.  
    0 + 1 <= (int)(Math.random() * 3) + 1 < 3 + 1  
    1 <= (int)(Math.random() * 3) + 1 < 4
